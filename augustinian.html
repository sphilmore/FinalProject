Chapter 3: The Augustinian Proof

Five Proofs of The Existence of God

The Augustinian Proof

A more formal statement of the argument
  1. There are three possible accounts for the existence of abstract objects ( numbers, possible worlds, universals, and propositions, and mathematical objects) realism, nominalism, and conceptualism.
  2. There are decisive arguments in favor of realism.
  3. There are insuperable objections against nominalism.
  4. There are insuperable objections against conceptualism.
  5. So, some versions of realism is true.
  6. There are three possible versions of realism: Platonic realism, Aristotelian realism, Scholastic realism
  7. If Platonic realism is true, then abstract objects exist in a “third realm” distinct from either the material world or any intellect
  8. If Aristotelian realism is true, then abstract objects exist only in the human mind or other contingently existing intellects.
  9. If Scholastic realism is true, then abstract objects exist only in contingently existing intellects but also in at least one necessarily existing intellect.
  10. There are insuperable objections against the claim that abstract objects exist in a “third realm” distinct from either the material world or any intellect.
  11. There are insuperable objections against the claim that abstract objects exist only in human or other contingently existing intellects.
  12. So, Aristotelian realism is not true.
  13. So, Scholastic realism is true.
  14. So, abstract objects exist not only in contingently existing intellects but also in at least one necessarily existing intellect.
  15. Abstract objects such as universals, propositions, numbers and other mathematical objects; and possible worlds are logically related to one another in such a way that they form an interlocking systems of ideas.
  16. The reasons for concluding that at least some abstract objects exist in a necessarily existing intellect also entail that this interlocking system of ideas must exist in a necessarily existing intellect.
  17. So, this interlocking system of ideas exist in at least one necessarily existing intellect.
  18. A necessarily existing intellect would be purely actual.
  19. There cannot be more than one thing that is purely actual.
  20. So,there cannot be more than one necessarily existing intellect.
  21. An intellect in which the interlocking system of ideas in question existed would be conceptually omniscient.
  22. So, the one necessarily existing intellect is conceptually omniscient.
  23. If this one necessarily existing intellect were not also omniscient in the stronger sense that it knows all contingent truths, then it would have unrealized potential and this not be purely actual.
  24. So, it is also omniscient in the stronger sense
  25. What is purely actual must also be omnipotent, fully good, immutable, immaterial, incorporeal, and eternal.
  26. So, there is exactly one necessarily existing intellect, which is purely actual, omniscient, omnipotent, fully good, immutable, immaterial, incorporeal, and eternal.
  27. But for there to be such a thing is just what it is for God to exist.
  28. So, God exists.
